# Escenarios de Simulación

## Parámetros base
- Sucursales: 1, 3, 5
- Ventanillas por sucursal: 2, 4
- Cajeros por sucursal: 1, 2
- Clientes simulados: 20, 50, 100
- Estrategias:
  - FIFO
  - Prioridad
  - Mixta

## Variables de medición
- Tiempo promedio de espera
- Transacciones procesadas por minuto
- Uso de CPU (en pruebas controladas)
