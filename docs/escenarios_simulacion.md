# Escenarios de Simulación

## Parámetros base
- Sucursales: 1, 2, 3, 5
- Ventanillas por sucursal: 1, 2, 4
- Cajeros por sucursal: 1, 2
- Clientes simulados: 20, 50, 100, 200
- Estrategias de atención:
  - FIFO (orden de llegada)
  - Prioridad (preferenciales primero)
  - Mixta (combinación de FIFO y prioridad)
- Duración de la simulación: configurable (segundos)

## Variables de medición
- Tiempo promedio de espera por cliente
- Tiempo promedio de servicio por cliente
- Total de transacciones procesadas
- Éxitos y fallos en transacciones
- Atenciones por ventanilla y cajero
- Fondos finales por sucursal
- Clientes restantes en cola
- (Opcional) Uso de CPU y rendimiento si se mide externamente
